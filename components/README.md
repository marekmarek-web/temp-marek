# Components

Struktura podle domény:

- `home/` — úvodní stránka
- `blog/` — články
- `forms/` — lead + subscriber formuláře
- `calculators/portal/` — kalkulačkové stránky
- `admin/` — CMS UI
- `layout/` — header, footer
- `ui/` — sdílené prvky (modal, dropdown, …)
- `auth/`, `consent/`, `analytics/`, `providers/` — podle názvu

Detailní mapa: `docs/repo-structure-guide.md`.
